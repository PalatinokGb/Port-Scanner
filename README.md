# Port-Scanner

# Multi-threaded Port Scanner

Um scanner de portos TCP rápido e eficiente, implementado com concorrência em Python através de `ThreadPoolExecutor`.

# Características
- Varredura multi-threaded para alta performance.
- Resolução automática de DNS para hostnames.
- Arquitetura limpa e focada na biblioteca padrão do Python.

#Como Executar
python Port_Scanner.py
